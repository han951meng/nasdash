标题：【nasdash】飞牛OS 硬件监控面板 v1.6.8：新增「检测新版本」功能

---

nasdash 是飞牛OS（fnOS）上的一款第三方硬件监控面板（fpk），实时展示你的 NAS 健康状态与硬件详情：

- 阵列卡状态（MegaRAID ROC 温度 / HBA 直通卡温度）
- 硬盘 SMART 健康与温度
- 系统资源：CPU、内存、温度、风扇、GPU、网卡
- 存储卷（mdadm）

近期更新：

🆕 **v1.6.8 —— 新增「检测新版本」功能**
- 面板自动检测 GitHub 最新 Release（带 6 小时缓存 + 5 秒超时，绝不卡页面）
- 发现新版本时，顶部弹出橙色更新横幅并附下载链接
- 右上角新增「🔔 检查更新」按钮，可手动立即刷新

✨ **v1.6.7 —— 飞牛桌面窗口型集成**
- 点飞牛桌面图标，直接在应用窗口内渲染，不再跳转外部浏览器，体验与系统应用一致

安装 / 升级：
1. 飞牛OS 应用中心 → 本地应用 → 上传 fpk，选择本包覆盖安装即可
2. 或从 GitHub 下载最新 .fpk：
https://github.com/han951meng/nasdash/releases/tag/v1.6.8
