# NAS硬件监控 v1.8.5

## 更新亮点
- **修复应用中心更新日志不显示**：此前在飞牛应用中心打开本应用，「更新日志」区域看不到版本更新内容。已修复——现在应用中心介绍页会稳定显示最近 3 个版本的更新要点。
- 本次为发布流程优化（补全应用中心描述的更新日志），**应用功能本身与 v1.8.4 完全一致**，无需重新配置。

## 说明
- 安装时自动安装依赖：smartmontools、lm-sensors、mdadm、Flask、dmidecode、i2c-tools。
- MegaRAID / HBA 直通卡用户无需手动下载 storcli，已随包内置并自动落地到 /opt/MegaRAID/storcli。
- 升级建议：可直接在飞牛应用中心卸载旧版后安装本 fpk；风扇标注、自定义名称、温控模式等配置位于 @appdata，重装默认保留。
