# nasdash v1.8.7

飞牛OS（fnOS）硬件监控面板更新。本版本重点强化了风扇控制，让「不同风扇跟同一温度、却各吹各的曲线」成为现实，并新增「平时彻底停转、高温才启暴力扇」的精确开关。

## 本版本更新

### ① 风扇控制新增「逐风扇温度联动控速」
每台风扇现在能**单独挑温度源**（硬盘聚合温度 / CPU / 主板），还能**单独设自己的控速曲线**。
- 比如两台硬盘风扇都跟同一组硬盘温度走，但一路用 30–45°C 曲线、另一路用 45–60°C 曲线——同温源也能各吹各的。
- 没单独设的风扇自动沿用原来的全局面板（硬盘温控 / 主板 CPU 温控）默认曲线，老配置照常生效、零破坏。

### ② 手动调速可设到 0%（真正停转）
原来手动调速最低只能拉到 10%，有些硬件在 10% 时风扇其实已经停了、却拉不下去、也显示不出「已停转」。本版把下限放开到 0%，并新增「停转」预设档，想让风扇彻底不转直接选它即可。

### ③ 重名风扇卡片不再串号
当不同芯片上有同名风扇（比如都叫「风扇 1」）时，旧版改名或隐藏单台会串到另一台。本版按「芯片 + 序号」区分每张卡片，改名/隐藏单台互不影响。

### ④ 硬盘温控新增「连续空闲 N 分钟即停转」
旧版只靠硬盘「休眠态」决定停风扇，可很多环境硬盘根本不报休眠态，导致风扇一直空转。本版新增一个选项：监控盘连续 N 分钟没有任何读写（按磁盘扇区读写计数判断）也停转风扇——与「盘休眠」二选一满足即停，N 默认 5 分钟、可设 1–120 分钟。

### ⑤ 逐风扇控速新增「低于开转温度即停转」开关
在「逐风扇温度联动控速」里，每台风扇多了一个勾选项：温度没到这台风扇的「开转温度」之前，强制把它停转（写 0%），而不是交还给主板自动控温。
- 专治「平时靠静音扇、高温才启暴力扇」的机器——把暴力扇开转温度设高、勾上这项，就能做到「不到那个温度它绝不转」。
- 只有明确读得到温度、且确实低于开转温度才强制停；读不到温度（传感器异常）会保守交还系统自动，不会在高温未知时把风扇熄火。

## 安装 / 升级

在飞牛OS 应用中心手动安装本 Release 的 `nasdash-release-v1.8.7.fpk` 即可（或从 GitHub Releases 下载后安装）。安装后桌面出现「NAS硬件监控」图标，点开即用。
